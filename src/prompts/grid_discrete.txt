CATEGORY: grid_discrete — discrete-state cellular automata on a square grid.

SPACE: type="grid". state_type = "Bool" | "Int" | "Symbol" | "String" | "Float64" (discrete via thresholds).
UPDATE: synchronous. The agent step writes the NEXT state to model.next_states[agent.id]; do NOT mutate agent.state directly.
RULES: agent_step="<your_fn!>"  model_step="default_model_step!"  initialization_rule="random" (or "center_filled" / "horizontal_line").
POPULATION: pop_density = { "state" = fraction, ... } (fractions sum to 1.0) OR pop_quantity = { "state" = N, ... }.
NEIGHBOURS: nearby_agents(agent, model).

BUILT-IN rules — use these and write NO _rules.jl when they fit:
- gol_step!        Game of Life (state_type="Bool"; properties: min_to_live, max_to_live)
- rps_step!        rock-paper-scissors (state_type="Symbol", values :rock/:paper/:scissors; property: threshold)
- schelling_step!  Schelling segregation (property: min_identical)

If you need a CUSTOM rule, write a _rules.jl containing ONLY step functions. In that file:
- NO using / import / module / @agent.
- signature: function my_step!(agent, model) ... model.next_states[agent.id] = new_value ... end

VISUALIZATION: variable_to_color="state"; color_scheme = { state = "colour", ... }; agent_shape = "rect" (or "circle", or a Dict mapping state → shape).

EXAMPLE — forest fire (Symbol state, custom rule):
FILENAME: fire.toml
```toml
[simulation]
model_name = "ForestFire"
seed = 42
[space]
type = "grid"
dimensions = [60, 60]
periodic = false
metric = "chebyshev"
[agents]
state_type = "Symbol"
[population]
pop_density = { "tree" = 0.7, "fire" = 0.05, "empty" = 0.25 }
[rules]
agent_step = "fire_step!"
model_step = "default_model_step!"
initialization_rule = "random"
[visualization]
filename = "output_videos/fire.mp4"
title = "Forest Fire"
variable_to_color = "state"
color_scheme = { tree = "darkgreen", fire = "orange", empty = "beige", ash = "gray" }
agent_shape = "rect"
agent_size = 10
framerate = 10
frames = 120
```
FILENAME: fire_rules.jl
```julia
function fire_step!(agent, model)
    if agent.state == :fire
        model.next_states[agent.id] = :ash
    elseif agent.state == :tree
        n_fire = count(nb -> nb.state == :fire, nearby_agents(agent, model))
        model.next_states[agent.id] = n_fire > 0 ? :fire : :tree
    else
        model.next_states[agent.id] = agent.state
    end
end
```
